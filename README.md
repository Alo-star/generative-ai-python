# 🧠 Python Generative AI

A collection of Python-based Artificial Intelligence and Machine Learning implementations, covering fundamental concepts, algorithms, and practical examples.

## 📌 Features

- 🤖 Hidden Markov Model (HMM)
- 🧠 Autoencoder using TensorFlow/Keras
- 📊 Data Analysis with NumPy & Pandas
- 📈 Data Visualization using Matplotlib
- 📝 Well-structured and beginner-friendly code

## 🛠️ Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- TensorFlow / Keras
- hmmlearn
- Scikit-learn

## 📂 Project Structure

```
python-generative-ai/

├── AutoEncoder/
```

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Alo-star/python-generative-ai.git
cd python-generative-ai
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run a Project

Example:

```bash
python hidden_markov_model.py
```

or

```bash
python autoencoder.py
```
